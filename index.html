ICSD Student Chromebook Audit

In-browser security assessment for managed student devices
About this tool: This page runs a battery of security tests from inside the browser. It checks what policies are being enforced, what URLs are reachable, what browser capabilities are available, and what common filter bypasses might work. All tests run locally in your browser - nothing is sent to any server. Load this page on a test student Chromebook while logged in as a test student account to audit the effective security policy on that device.
