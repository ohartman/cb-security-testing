ICSD Student Chromebook Audit

Policy enforcement and browser capability assessment (Linewize-safe)
Linewize-safe design: This version is designed to run safely alongside Linewize Connect. It makes NO requests to URLs Linewize would consider bad - no proxy sites, no DoH endpoints, no CIPA probes. All tests either (1) make no network requests at all, (2) only reach Google and district services that are already allowlisted, or (3) are manual checks you perform yourself. The goal is to produce useful findings without triggering Linewize's block page redirect.
What this tests: The things content filtering cannot tell you - Chrome policy enforcement, browser hardware APIs, WebRTC behavior, and a comprehensive manual checklist for the physical verification work. Content filter effectiveness is verified separately in your Linewize dashboard, not here.
State persistence: Test results are saved to localStorage, so if something causes the page to reload or navigate away, click "Run All Tests" again to continue. Completed results will be restored.
